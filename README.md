## Getting Started

1. Clone this repository

```bash
git clone https://github.com/stineknutsen/portfolio-2.git
```

2. Install dependencies

```bash
npm install
```

3. Available scripts

```bash
npm run dev
```

```bash
npm run build
```

```bash
npm run preview
```

The app should now run at localhost
